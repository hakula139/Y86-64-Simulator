# Y86-64 Simulator UI

An interface for Y86-64 Simulator built in Javascript.

Here's a [Demo](https://sim.hakula.xyz).

## Getting Started

### 0. Prerequisites

- [Node.js](https://nodejs.org/en/download) 12 or above

### 1. Building

```bash
npm install
npm run build
```

### 2. Usage

```bash
npm run start
```
